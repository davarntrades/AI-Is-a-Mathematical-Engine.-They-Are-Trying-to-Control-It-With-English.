## The Paper That Started Everything

In 2017, a team at Google Brain published a paper titled *Attention Is All You Need*.

It introduced the transformer architecture — the foundation of every major AI system running today. GPT. Claude. Gemini. Llama. All of them. Transformer architecture. All of them descended from this paper.

The paper is pure mathematics.

```
Attention( Q, K, V ) = softmax( QKᵀ / √dₖ ) · V

Q = query matrix
K = key matrix
V = value matrix
dₖ = dimension of key vectors
softmax = probability normalisation function
```

This is not a metaphor. This is what runs every time you send a message to an AI. A matrix query is computed against a matrix of keys. The result is scaled. A softmax function converts it to a probability distribution. That distribution weights a value matrix. The output is a weighted geometric transformation in high-dimensional space.

There is no English in this equation. There is no language in this equation. There is geometry. There is linear algebra. There is a mathematical operation that transforms vectors through dimensional space.

The researchers who published this knew exactly what they were building. A mathematical engine. They described it in the only language that could describe it accurately. Mathematics.
